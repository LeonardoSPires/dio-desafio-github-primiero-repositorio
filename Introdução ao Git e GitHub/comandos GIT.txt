Comandos GIT:

DIR : mostra todas as pastas.

del + nome da pasta: apaga todo conteudo dentro da pasta.

rmdir + nome da pasta: apaga a pasta.

echo + nome do documento: cria um documento dentro da pasta.

CLS: limpa a tela.

mkdir +nome da pasta: cria uma nova pasta.

openssl sha1 = nome do arquivo : cria a
 criptografia de 40 digitos.

como enviar o repositorio local para o remoto:
1- Vá até a pasta e abra o git bash

2- Depois de um git init

3- de um git add *

4- git commit -m "Commit inicial"

git remote add origin + link do github.

Informa a lista de repositorio remoto
git remote -v

saber se ha alguma pendencia no repositório.
git status

empurra o código para o github
git push origin master
